# FoxGear Netz Artz

Проект FoxGear Artz - лендинг-страница для приобретения компьютерного стола переменной высоты. В качестве основы взят англоязычный сайт ОЕМ-производителя, но верстка блоков и страницы в целом выполнено независимо от исходника.

Переведена вся графика для всех вариантов модели и реализовано переключение цветов кресел при помощи селектора.

## Технологии

В этом проекте использованы следующие технологии:

- **HTML/CSS** — для создания и стилизации интерфейса
- **JavaScript** — для интерактивных элементов
- **JQuery** — для создания каруселей
- **OwlCarousel 2** — для создания каруселей
- **PHP/mail()** отправка уведомлений о размещении заявки на сайте

- **Адаптивный дизайн:** любое разрешение браузера, мобильное меню.
